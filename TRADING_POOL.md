## 三、风控规范

### 硬止损规则
- DOT: 止损 2% 或 1.5 ATR（取大者）
- AVAX突破: 追踪止损5%（吉总指定）
- ETH/BTC: 止损 2%
- 任意策略最大回撤 > 30% → 全仓停止交易

### 市场环境过滤
- EMA20 < EMA50 < EMA200 → 熊市 → 仓位降至20%
- EMA20 > EMA50 > EMA200 → 牛市 → 正常仓位
- MA200走平 → 震荡 → 观望

### 信号优先级
1. DOT RSI<35 → 立即执行（熊市也有效）
2. AVAX 突破30日/20日高点 → 执行（任何环境）
3. ETH RSI<45 → 牛市环境才执行
4. BTC RSI<35 → 牛市环境才执行

## 六、策略开发强制规则

### 必须遵守
1. **必须包含非价格类指标**：RSI/ADX/MACD/成交量/资金费率等
2. **必须通过四层验证**：历史回测 → Walk-Forward → 极端行情 → 蒙特卡洛
3. **必须做熊市压力测试**：2022熊市（含FTX）窗口必须正收益
4. **不允许纯价格行为策略**：K线形态/支撑阻力必须配合指标过滤

### 禁止
- 纯价格行为策略（如仅靠"趋势分类"决定持仓）
- Walk-Forward < 100%通过的策略（除非有特殊理由）
- 2022熊市窗口亏损 > 30%的策略
- 没有经历过完整牛熊周期的策略
