# Kronos Trading AI - 系统文档 v6

## 核心理念

**不是预测机器，是执行系统。**

交易的核心不是"预测对方向"，而是"用系统化的方法执行已验证的规律，并在执行中不断学习"。

**"老天吃饭"是死路一条** — 等完美信号一年交易2-3次，永远无法验证和进步。

## 策略体系

### 核心策略：BTC RSI高频（STR-001改）

| 项目 | 值 |
|------|-----|
| 买入条件 | RSI < 35 |
| 卖出条件 | RSI > 65 或 持满7天 或 止损2% |
| 预期年交易 | 18笔 |
| 预期胜率 | 57% |
| 预期PF | 1.99 |
| 预期年化 | +64.8% |
| 数据验证 | 2024-2026（2年），2023年样本外验证 |

### 备选策略：BTC布林趋势（STR-003）

| 项目 | 值 |
|------|-----|
| 买入条件 | 突破布林上轨 |
| 卖出条件 | 价格跌破布林中轨 或 止损1.5ATR |
| 预期年交易 | 8笔 |
| 适用环境 | 牛市(BULL) |

### 合约策略（仅熊市）

| 市场环境 | 策略 | 逻辑 |
|----------|------|------|
| 熊市(BEAR) | BTC RSI双向合约 | RSI<30做多，RSI>70做空 |
| 牛市(BULL) | ETH BB双向合约 | 突破上轨做多，跌破下轨做空 |

## 关键认知升级

**今天最重要的发现不是哪个策略有效，而是验证方法论：**

1. **回测是起点，不是终点** — 必须有样本外验证和步行前向测试
2. **单一参数优化不够** — 需要多币种、多市场、多参数同时验证
3. **频率 vs 质量 trade-off** — 高频不等于好交易，放宽阈值等于放弃策略核心优势
4. **BTC是唯一的** — 其他币种不能用同样的策略

**系统已经完成从"研究"到"执行"的转变。**
