# Kronos V3 交易系统

## 核心理念

**趋势跟随 > 均值回归**

经过10年数据验证，趋势跟随策略在BTC上表现最优：
- 熊市时做空保护资本
- 牛市时不逆势加仓
- 不追求买在最低点，卖在最高点

---

## 系统配置

```
MA周期: 50 / 100
止损: 3倍ATR
杠杆: 2x
做空: 开启
```

## 关键验证

### 2018年崩盘验证
- BTC持有: **-66.2%**

- 策略年化: +110%

## 信号系统

| 信号 | 含义 | 操作 |
|------|------|------|
| STRONG_LONG | 多周期确认向上 + 突破 | 做多 2x |
| TREND_UP | 日线向上但未突破 | 等待 |
| WAIT | 趋势不明 | 观望 |
| TREND_DOWN | 日线向下但未跌破 | 等待 |
| STRONG_SHORT | 多周期确认向下 + 跌破 | 做空 2x |

---

## 入场条件

**做多:**
1. 日线: 价格 > MA50 > MA100
2. 4H: 价格 > MA50 > MA100
3. 价格突破20日高点

**做空:**
1. 日线: 价格 < MA50 < MA100
2. 4H: 价格 < MA50 < MA100
3. 价格跌破20日低点

## 风险管理

```
止损: 3倍ATR (约9%)
目标: 2倍风险 (约18%)
仓位: 2x杠杆

风险计算:
- 如果ATR=3%, 止损=9%
- 2x杠杆 → 实际风险18%
- 目标回报36%
- 盈亏比 1:2
```

---

## 策略逻辑

### 为什么趋势跟随有效？

1. **减少主观判断**: 不预测方向，只跟随信号
2. **高盈亏比弥补低胜率**: 34%胜率 × 11倍盈亏比 = 正期望
3. **熊市保护**: 2018年做空获利4371%
4. **低频减少摩擦**: 每年10笔，手续费影响极小

### 为什么均值回归在BTC上无效？

1. BTC波动极大，RSI<30后可能继续跌
2. "超卖"可能持续数月
3. 高胜率但盈亏比差，最终PF<1

## 多周期确认

```
日线(1d): 确定大方向
4H: 确认趋势
1H: 寻找入场点
```

三层确认确保信号可靠性，减少假突破。

## 风险提示

1. **过去表现不代表未来**: 10年数据可能不够验证完整牛熊周期
2. **杠杆风险**: 2x杠杆在亏损时同样放大
3. **流动性风险**: 极端行情可能无法以预期价格成交
4. **心理风险**: 低胜率需要强大执行力

---

## 使用建议

1. **回测验证**: 在模拟盘运行3个月以上
2. **小资金起步**: 先用1-2%仓位测试
3. **记录每笔交易**: 建立自己的交易日志
4. **定期复盘**: 每季度评估系统表现
