## 核心原则

> 预测方向是最愚蠢的事。捡钱才是正道。

**能做：** 批量回测、统计概率、扫描结构性失效、自动化执行、链上数据建模
**不能做：** 解读新闻叙事、判断市场情绪、预测黑天鹅、处理极端流动性枯竭、主观干预交易

**铁律：** 任何需要"理解叙事"的决策交给人类；任何可以用数据量化的决策全部交给系统。

## 第一阶段：基础能力搭建（4月18日-4月25日）

### P0-1: 实盘日志系统（4月20日前上线）
数字货币专属字段：
- 杠杆倍数、保证金率、爆仓价格
- 下单深度、滑点率、成交耗时
- 同期全市场爆仓量、资金费率
- 链上大额转账（>1000 BTC）记录

核心统计：
- 区分"策略亏损"、"滑点亏损"、"杠杆爆仓亏损"
- 区分不同时间段（凌晨/白天/欧美盘）的执行滑点差异
- 爆仓峰值前后策略的表现变化

### P0-2: 事件日历与回测引擎（4月23日前）
聚焦数字货币独有事件，按优先级：
1. 核心周期：BTC减半、以太坊升级（坎昆/德班）、季度期货/期权交割
2. 项目事件：代币解锁、主网上线、L2空投、大额回购、审计结果
3. 行业事件：ETF审批/扩容、监管政策、交易所上线/下架、稳定币脱钩
4. 链上事件：大额转账集中、矿工抛压峰值、Gas费异常

第一个目标：回测2020年以来所有季度期权交割日（最后一个周五）前后3天的ETH价格波动规律

### P1-3: 多资产相关性矩阵（4月25日前）
- 赛道相关性：BTC/ETH → DeFi → AI → Meme → LSD
- L1/L2相关性：ETH → Arbitrum → Optimism → Base
- 跨周期相关性：周线方向 → 4小时趋势 → 15分钟入场

## 第二阶段：核心edge能力建设（4月26日-5月31日）

### P0-1: 第一个事件驱动策略（5月10日前上线）
推荐优先级：
1. **季度期权到期钉住效应**：到期前24小时，价格钉在最大痛点；到期后波动率爆发
2. **代币解锁前抛压效应**：解锁前7天平均下跌5-8%，解锁后1-3天反弹
3. **资金费率极端反转**：连续3天>0.15%或<-0.15%时反向开仓，持有24h

仓位规则：单策略≤2%总资金，最高2x杠杆

### P0-2: 市场结构性失效扫描器（5月20日前）
24小时不间断扫描：
1. 跨交易所期现价差套利：Binance vs OKX 同一币种永续合约价差>0.3%
2. 资金费率套利：费率>0.2%时做空永续+做多现货；<-0.2%时反向
3. 期权波动率套利：同一行权价IV差>10%
4. L2与主网价差套利：L2代币与主网代币价差>1%

### P1-3: 链上数据预警系统（5月31日前）
监控指标：
- BTC/ETH大额转账（>1000 BTC / >10000 ETH）
- 交易所净流入/流出量
- 稳定币发行量变化
- 矿工抛压（矿工地址余额变化）
- 合约未平仓量与持仓量变化

预警规则：
- 交易所净流入>20日均值3倍 → 下跌预警
- 稳定币发行量单日增长>5% → 上涨预警
- 合约未平仓量超历史峰值 → 爆仓风险预警

## 第三阶段：系统进化（6月1日后）

### 多策略投资组合优化
- 目标：最大回撤从30%降至20%以下，保持年化收益
- 熊市：提高套利/空头策略权重，降低多头策略权重

### 自动化策略发现与淘汰
- 自动扫描新事件、新链上指标、新市场结构
- 四层验证（重点2022熊市+2021牛市）
- 连续3个月表现不佳 → 自动淘汰

### 黑天鹅防御系统
- 1小时爆仓量>10亿美元 → 自动平仓所有仓位
- USDT脱钩>0.5% → 自动平仓所有仓位
- 交易所API宕机 → 自动切换备用交易所
